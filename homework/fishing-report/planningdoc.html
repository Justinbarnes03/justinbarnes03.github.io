Wireframe Sketches

My Web Pages

The homepage and North Idaho Report contain Javascript and json

  • HomePage: This is page has links to each region's fishing report. It also conatins my content video of me releasing a brown trout. The Homepage contains my json information. The Json is placed in the region containers and places the river names in each region.
  • Northern Idaho Report: This page give a report of 4 different river in North Idaho. The Json Is also on this page. It is used to place a report in every river Report section. Each river section highlights what flies are working for each region.
  • Eastern Idaho Report: This page gives a detailed report of the Snake River, Henry's Fork, Big Lost River, and the Blackfoot River. There is a highlighted fly section telling the user what flies to use. There is also a detailed report of the fishing on the given river.
  • Western Idaho Report: This page gives a detailed report of the Boise River, Owyhee River, Rapid River, and the Payette River. There is a highlighted fly section telling the user what flies to use. There is also a detailed report of the fishing on the given river.
  • Centeral Idaho:This page gives a detailed report of the Middle Fork of the Salmon River, Selway River, Big Wood River, and the Silver Creek. There is a highlighted fly section telling the user what flies to use. There is also a detailed report of the fishing on the given river.

Usability Concepts

  • I will keep my load time as low as possible by Sizing image appropriately so evrything loads quickly.
  • I will use contrating colors for the Text so it is visible and easy to read.
  • I will use a font that is easy to read and a font size that is easy on the eyes.
  • I will use appropriate alt tags That can be identified by screen readers.
  • The site title and headings will be clearly visible.
  • Make a user friendly nav bar that is easy to navigate. The nav will change colors when clicked.
  • There will be a trout logo in the top left part of the header.
  • Information will be short and concise and listed when possible, for easier memorization.
  • Each page will have a title that is easy to identify.
